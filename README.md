# Kalkulator PAD_Z4
Mateusz Stryjeński
Projekt wykonany w technologii WPF (.NET 8) w ramach przedmiotu Programowanie Aplikacji Desktopowych.

## Funkcjonalności

* dodawanie
* odejmowanie
* mnożenie
* dzielenie
* procenty
* pierwiastkowanie
* odwracanie liczby (1/x)
* potęgowanie (x^y)
* ponawianie operacji "="
* resetowanie buforów

## Technologie

* C#
* WPF
* .NET 8
